# Laboratory 1: ***"warm-up"***

## Travel Guide

### Input
- Undirected graph G = (V, E).
- Function c: E -> N assigning weights to edges.
- Special vertices 's' and 't'.

### Problem
Find path from s to t such that the least weight on this path
is as big as possible.

### Output
- flow via found path.
